# Playbook: ICT Fair Value Gap Entry

## מהו FVG
Gap בין wick של נר אחד ל-body של נר שלישי — "חלל" שהשוק לא מילא.
המוסדיים רוצים למלא אותו → חזרה ל-FVG = הזדמנות כניסה.

## תנאי כניסה (Bullish FVG)
1. מבנה שוק Bullish בTimeframe גבוה (4H/Daily)
2. FVG נוצר בתנועה Bullish חזקה (Impulse)
3. מחיר חוזר ל-FVG בנפח נמוך (Retracement)
4. מחיר מגיב בתוך ה-FVG (Candle Reaction)

## תנאי כניסה (Bearish FVG)
1. מבנה שוק Bearish בTimeframe גבוה
2. FVG נוצר בתנועה Bearish חזקה
3. מחיר חוזר ל-FVG בנפח נמוך
4. מחיר מגיב בתוך ה-FVG

## כניסה
- כניסה בחצי ה-FVG או בגבול הרלוונטי
- אישור נר Confirmation (Pin Bar / Engulfing)

## Stop Loss
מעל/מתחת ל-FVG כולו (לא לחצות אותו)

## Take Profit
- TP1: High/Low הקודם (Liquidity)
- TP2: FVG הבא בכיוון

## Kill Zones (עדיפות)
- London Open: 10:00-12:00 (שעון ישראל)
- NY Open: 16:30-18:30 (שעון ישראל)
- Asian Midnight: 01:00-03:00 (שעון ישראל)
